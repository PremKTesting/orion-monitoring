# A/B Testing script
**USAGE:** `python3 ab_comparison.py 	-a <a_report_directory> 
										-b <b_report_directory> 
                                        -o <output_file>`

**EXAMPLE:** `python3 ab_comparison.py 	-a '/home/user/r201_d130_o540' 
										-b '/home/user/r201_d130_o550' 
                                        -o '/home/user/output.json'`
